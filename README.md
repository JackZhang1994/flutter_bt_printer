# flutter_bt_printer

精臣蓝牙打印插件


#### 通道

flutter_bt_printer

#### 方法

| 方法名 | 含义   | 参数                                                         |
| ------ | ------ | ------------------------------------------------------------ |
| init   | 初始化 |                                                              |
| print  | 打印   | asset_number：资产编号；asset_name：资产名称；asset_model：规格型号 |
| quit   | 退出   |                                                              |
